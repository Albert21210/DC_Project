# REST API - Компьютерный магазин «Pixel»

Интерактивная документация Swagger: `http://localhost:8000/docs`
ReDoc: `http://localhost:8000/redoc`

## Эндпоинты

### Система
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/health` | Проверка состояния сервиса |

### Товары `/api/products`
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Все товары с категорией и поставщиком |
| GET | `/available` | Только товары в наличии |
| GET | `/low-stock?threshold=10` | Критические остатки |
| GET | `/search?q=Intel` | Поиск по названию |
| GET | `/by-category/{id}` | Фильтр по категории |
| POST | `/restock` | Пополнение склада |

### Заказы `/api/orders`
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/?status=&skip=0&limit=50` | Список заказов |
| GET | `/unpaid` | Неоплаченные заказы |
| GET | `/{id}` | Детали заказа с позициями |
| POST | `/` | Создать заказ |
| POST | `/{id}/pay` | Подтвердить оплату |
| POST | `/{id}/cancel` | Отменить заказ |

### Аналитика `/api/analytics`
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/summary` | Сводные показатели |
| GET | `/top-products?limit=3` | ТОП продаваемых товаров |
| GET | `/monthly-revenue?days=30` | Выручка за период |
| GET | `/order-sums` | Сводка всех чеков |
| GET | `/employee-efficiency` | Рейтинг сотрудников |
| GET | `/revenue-by-category` | Выручка по категориям |

### Клиенты, Сотрудники, Поставщики, Категории
| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/customers/` | Список клиентов |
| POST | `/api/customers/` | Добавить клиента |
| GET | `/api/employees/` | Список сотрудников |
| GET | `/api/suppliers/` | Список поставщиков |
| GET | `/api/categories/` | Категории с кол-вом товаров |

## Коды ответов

| Код | Значение |
|-----|----------|
| 200 | Успех |
| 400 | Ошибка валидации / бизнес-правила |
| 404 | Ресурс не найден |
| 500 | Внутренняя ошибка сервера |
